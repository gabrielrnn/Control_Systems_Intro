

# Sobre a intenção desse repositório
Depois de um tempo estudando controle, comecei a ter a vontade de criar um projeto em que pudesse aplicar os conhecimentos que via em livros na prática para poder testar o que dá certo (e o que não dá). Basicamente, esse repositório consistirá de algumas tentativas de simular sistemas dinâmicos simplificados em busca de testar metodologias de controle que aprendo com o tempo, de forma que eu possa testar alguns insights e tirar dúvidas na prática sobre essas metodologias. Essa também é uma forma que encontrei de praticar um pouco minhas habilidades de programação de maneira um pouco mais orgânica do que simplesmente fazer exercícios no LeetCode. 

Considerando esse escopo, pretendo escrever essa documentação, pelo menos a princípio, de maneira bem direta. No entanto, é possível que eu vá adicionando seções mais detalhadas conforme o projeto fique mais completo.
